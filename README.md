# 河北金融学院公众号 NER 实体识别系统

一个针对**河北金融学院官方微信公众号**文章进行实体识别、可视化与文章概括的交互式系统。支持文本、图片 OCR、URL 抓取、文件上传、批量处理等多种输入方式。

## 功能特点

- **NER 引擎**：基于 `jieba` 词性标注 + 河北金融学院领域词典 + 规则，零依赖，离线可用，适合所有部署环境。
- **大模型生成式摘要**（可选）：接入智谱 GLM-5.2 API，可切换为 AI 生成式文章总结（需在侧边栏配置 API Key）。
- **针对性实体类型**：人名、组织机构、地点、时间、活动/事件、专业/课程、职务/职称、金额、百分比。
- **多种输入方式**：粘贴文本、图片 OCR、微信公众号 URL 抓取、单文件上传、批量文件处理。
- **三大前端分区**：
  1. **实体识别**：原文高亮 + 实体列表 + 类型筛选 + CSV/JSON 导出。
  2. **可视化分析**：实体类型分布、高频实体 Top 10、**词云图**、**实体共现网络**、共现矩阵。
  3. **文章总结**：摘要（抽取式/生成式）、关键词、主题分类、关键信息汇总。
- **批量处理**：一键处理多个 txt/html 文件，导出汇总 CSV。

## 安装

```bash
cd "d:/HF/二学期/自然语言处理/结课"
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

然后打开浏览器访问 `http://localhost:8501`。

## 配置智谱 GLM API（可选）

在 Streamlit 侧边栏"大模型总结"栏中填入智谱 GLM API Key，即可启用 AI 生成式摘要。
也可在 Streamlit Cloud 的 Secrets 中添加：

```toml
GLM_API_KEY = "你的API_Key"
```

## 项目结构

```text
├── app.py            # Streamlit 交互前端（三标签页 + 批量处理）
├── ner.py            # NER 引擎（jieba 领域词典 + 规则）

├── visualizer.py     # 词云图、共现网络、共现矩阵
├── requirements.txt  # 依赖
└── README.md         # 使用说明
```

## 使用说明

1. 在左侧选择输入方式，把文章文本/图片/URL/文件放入系统。
2. 点击 **"开始识别"**。
3. 在 **"实体识别"** 标签页查看原文高亮和实体列表。
4. 切换到 **"可视化分析"** 查看统计图表、词云图和共现网络。
5. 切换到 **"文章总结"** 查看摘要、关键词和主题分类。
6. （可选）在侧边栏填入智谱 GLM API Key，启用大模型生成式摘要。
7. 使用 **批量文件处理** 模式可一次处理多个文件并导出汇总。

## 注意事项

- 首次运行会自动加载 jieba 领域词典。
- 词云图显示效果依赖于系统字体，Linux 环境需安装中文字体（已通过 `packages.txt` 配置）。
- URL 抓取可能受微信公众号反爬或登录限制，若失败可改用"粘贴文本"或"上传图片"。
- 抓取内容仅供学习/研究使用，请遵守相关网站的 robots 协议与版权规定。
- 智谱 GLM API 需自行申请，API Key 请妥善保管，不要上传至公开仓库。
