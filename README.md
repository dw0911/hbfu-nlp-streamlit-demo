# 河北金融学院公众号 NER 实体识别系统

一个针对**河北金融学院官方微信公众号**文章进行实体识别、可视化与文章概括的交互式系统。支持文本、图片 OCR、URL 抓取、文件上传、批量处理等多种输入方式。

## 功能特点

- **双引擎 NER**：
  - **高级模型**：基于 spaCy `zh_core_web_sm` 中文模型。
  - **离线备选**：基于 `jieba` 词性标注 + 河北金融学院领域规则。
  - 自动检测 spaCy 是否可用，失败时无缝回退到 jieba。
- **针对性实体类型**：人名、组织机构、地点、时间、活动/事件、专业/课程、职务/职称、金额、百分比。
- **多种输入方式**：粘贴文本、图片 OCR、微信公众号 URL 抓取、单文件上传、批量文件处理。
- **三大前端分区**：
  1. **实体识别**：原文高亮 + 实体列表 + 类型筛选 + CSV/JSON 导出。
  2. **可视化分析**：实体类型分布、高频实体 Top 10、**词云图**、**实体共现网络**、共现矩阵。
  3. **文章总结**：摘要、关键词、主题分类、关键信息汇总。
- **批量处理**：一键处理多个 txt/html 文件，导出汇总 CSV。

## 安装

```bash
cd "d:/HF/二学期/自然语言处理/结课"
pip install -r requirements.txt
```

### 安装 spaCy 中文模型（可选）

如果希望使用高级模型，请执行：

```bash
python -m spacy download zh_core_web_sm
```

若下载失败或不需要，系统会自动使用 jieba 离线规则作为备选。

## 运行

```bash
python start_server.py
```

然后打开浏览器访问 `http://localhost:8501`。

或者直接在终端运行：

```bash
streamlit run app.py
```

## 使用说明

1. 在左侧选择输入方式，把文章文本/图片/URL/文件放入系统。
2. 选择 NER 引擎（默认自动选择，优先 spaCy）。
3. 点击 **“开始识别”**。
4. 在 **“实体识别”** 标签页查看原文高亮和实体列表。
5. 切换到 **“可视化分析”** 查看统计图表、词云图和共现网络。
6. 切换到 **“文章总结”** 查看摘要、关键词和主题分类。
7. 使用 **批量文件处理** 模式可一次处理多个文件并导出汇总。

## 注意事项

- 首次运行会自动加载 jieba 词典；若使用 spaCy 模型，请确保模型已下载。
- 如果 spaCy 模型下载失败，选择“自动选择”或“jieba 离线规则”即可直接使用，无需联网。
- 词云图显示效果依赖于系统字体，Windows 通常可正常显示中文；若显示为方框，可安装中文字体或修改 `visualizer.py` 中的 `font_path`。
- URL 抓取可能受微信公众号反爬或登录限制，若失败可改用“粘贴文本”或“上传图片”。
- 抓取内容仅供学习/研究使用，请遵守相关网站的 robots 协议与版权规定。
